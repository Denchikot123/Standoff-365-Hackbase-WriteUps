# Web-2-1 Path Traversal Exploit Script

## ⚙️ Техническое описание и стек

Скрипт разработан для автоматизации эксплуатации уязвимости **Path Traversal** в категории **BootCamp** на ИБ-полигоне **Standoff 365 Hackbase**. Он отправляет запрос на чтение файла из параметра **?file=** в пейлоадк. Скрипт возвращает полученный ответ сервера.

### 🧰 Инструменты и библиотеки
* **Язык**: Python
* **Пакеты**:
  * `Requests` — работа с вэб-запросами.

### 🔄 Процесс работы скрипта

```mermaid
graph LR
    A["⚙️ Старт<br>(main)"] --> B["📝 Формирование payload<br>(file=/etc/pt.flag)"]

    D -->|Успешно| E["📄 Вывод содержимого<br>(print flag)"]
    D -->|Ошибка сети| F["❌ Вывод ошибки<br>(print Exception)"]
```

1. **Установите зависимости**:
   ```bash
   pip install requests
   ```
2. **Запустите скрипт**:
   ```bash
   python Web-2-1-autocomplete.py
   ```
