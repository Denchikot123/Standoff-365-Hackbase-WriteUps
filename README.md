# Standoff-365-Hackbase-WriteUps

## Суть работы

В данном репозитории будут представлены отчеты/разборы по 3 категориям обучающего курса **HackBase** из платформы **Standoff 365**

## Категории HackBase:
| Type | Level | Status |
|--------|----------|-------|
| **BootCamp** | Beginner/Easy | **🟡 In process (3 / 31)** |
| **Standalone** | Medium | ⚪ Upcoming |
| **Industry** | Hard | ⚪ Upcoming |

> 💡 **Обозначение статусов:**
> * 🟢 `Completed` — Выполнено
> * 🟡 `In Progress (X / X)` — В процессе (Сколько выполнено / Сколько всего)
> * ⚪ `Upcoming` — Не выполнено / Скоро

### Структура репозитория:

Сами отчеты сортированы по папкам с названием Категории HackBase и их под типу в самой категории

```text
📁 [Категория_HackBase]/
└── 📁 [Name_Subtype]/
    ├── 📄 README.md              # Отчет по лабе/хосту (name X-X)
    └── 📁 [SCRIPT-NAME]          # Папка со скриптом автоматизации (script-name X-X)
        ├── 📄 exploit.py
        └── 📄 README.md          # Пояснение кода
```

К каждому скрипту будет пояснение сути его работы, в основном будет использоваться ЯП - **Python**
